## Spec

Added extended flag notation for less common use cases.

- `nargs` amount of arguments consumed

```yaml
name: command
commands:
  - name: extended
    flags:
      --nargs-two=: {description: consumes two arguments, nargs: 2}
      --nargs-any=: {description: consumes multiple arguments, nargs: -1}
```
