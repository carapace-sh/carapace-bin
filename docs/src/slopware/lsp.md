# LSP

An experimental [Bash Language Server][lsp] with support for [Carapace].

![](./lsp/bash-language-server.cast)

Place the single binary release in your [PATH] and follow the [original instructions](https://github.com/bash-lsp/bash-language-server).

[lsp]:https://github.com/carapace-sh/bash-language-server
[Carapace]:https://carapace.sh
[PATH]:https://github.com/bash-lsp/bash-language-server
