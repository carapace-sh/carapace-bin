# Skills

[Skills] extend agent capabilities. Simply place them into your [skills folder].

```sh
crush
├── crush.json
└── skills
    ├── carapace-integrate
    │   └── SKILL.md
    ├── carapace-scrape
    │   └── SKILL.md
    └── carapace-spec
        └── SKILL.md
```
