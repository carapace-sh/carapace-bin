# Carapace

Use the `carapace` MCP server to determine potential argument of a command.

## Examples
- `["git", "-"]}` complete shorthand and longhand flags of `git`
- `["git", "checkout", "--"]}` complete longhand flags of `git checkout`
- `["git", "show", ""]}` complete positional arguments of `git show`
